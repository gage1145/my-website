This tool is written in Python and uses several methods of image manipulation using the pillow library. First the image is resized based on the width parameter. A magic number of 2.2 is used to account for the difference in height and width of the monospace font. def resize_image(image, new_width): image = ImageOps.exif_transpose(image) width, height = image.size ratio = height / (2.2 * width) new_height = int(new_width * ratio) return image.resize((new_width, new_height)) Next contrast is applied. def change_contrast(img, level=100): factor = 259*(level+255)/255*(259-level) def contrast(c): return 128 + factor * (c-128) return img.point(contrast)
